** Other areas not shown

- *Scroll bar* — disabled via ~vertical-scroll-bars~ in early-init
- *Window divider* — thin line between split windows (configurable via ~window-divider-mode~)
- *Internal border* — padding around the entire frame edge (~internal-border-width~)
- *Child frame* — floating frames used by some packages (posframe, corfu)
